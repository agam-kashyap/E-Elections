About

This website is for conducting elections. People waste a lot of time waiting in lines to vote. With this website, your problems have been solved!

Users can use the username and password provided to them to login and cast the vote.

Managers can use the secret id provided to them to add new voters. The results of the vote can be seen »here«.

Candidates can login with the username , password and the key provided to add information about themselves which will be shown to the users.

Candidates can add and update information about them and links to their social profile pages.
